# 🌊 seb_Atlantis v3.0

Interface unifiée de gestion IA — Desktop + PWA.

## Fonctionnalités

- **💬 Chat IA** — Conversation streaming via OpenRouter (multi-modèles)
- **📁 Fichiers** — Explorateur avec arbre, recherche, prévisualisation
- **🖥️ VPS** — Gestion multi-serveurs avec terminal simulé
- **🧠 Mémoire** — Mémoire étendue avec tags et recherche
- **⚙️ Réglages** — Clé API, modèles, thèmes, export/import

## Stack

- React 19 + TypeScript
- Vite 7
- Zustand (state management)
- react-router-dom 7
- vite-plugin-pwa (mode standalone installable)
- Design : dark theme custom (variables CSS)

## Démarrage

```bash
npm install
npx vite dev        # Dev server
npx vite build      # Build production
npx vite preview    # Preview du build
```

## Accès

- Dev : http://localhost:5173
- Preview build : http://localhost:7000
